This folder contains the workspaces and all their variables for the parameterizations of the
Lotka-Volterra model that are found in the final REU write up.

final_dram.mat: contains the variables from the DRAM parameterization
	- generates figures in figures/final_dram


final_mh.mat: contains the variables from the Metropolis parameterization
	- generates figures in figures/final_mh
