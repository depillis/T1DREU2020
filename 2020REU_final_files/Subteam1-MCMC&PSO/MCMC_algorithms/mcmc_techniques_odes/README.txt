mcmc_techniques_odes
This folder contains R code from the article
"MCMC Techniques for Parameter Estimation of ODE Based Models in Systems Biology"
Authors: Gloria I. Valderrama-Bahamóndez and Holger Fröhlich

This paper compares a variety of MCMC algorithms.
The comparison included Metropolis-Hastings, parallel tempering MCMC, adaptive MCMC, and parallel adaptive MCMC.
