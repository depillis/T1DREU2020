T1D

This folder contains all the files and folders for the Type 1 Diabetes parameter estimation
using PSO and MCMC techniques. 
=====================================================================================
1) Data_cleaning contains folders for mouse data

2) MCMC contains the tutorial for parameter estimation using MCMC methods

3) Method_comparison contains functions and figures for comparing parameter estimation         techniques (MCMC, PSO, and UKF)

4) PSO contains the tutorial for parameter estimation using PSO methods
