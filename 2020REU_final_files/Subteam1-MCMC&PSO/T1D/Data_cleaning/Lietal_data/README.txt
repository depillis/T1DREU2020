Lietal_data

This folder contains subfolders of with different data cleaning methods for using the Li et al data.

Averaged_acute contains data and instructions for averaging the acute mouse data

PostOnsetEstimation was developed by the UKF subteam and contains their version of determining diabetes onset and averaging data

Raw_data contains the original .csv files for all 11 mice in the Li et al data set. It also contains several T1D ODE functions however these were not used in our tutorial
