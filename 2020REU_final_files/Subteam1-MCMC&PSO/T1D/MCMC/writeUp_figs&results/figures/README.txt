figures

This folder contains the figures (.fig and .png files) from the HMC REU Summer 2020 final report.
The folders are divided by the data with which the DRAM parameterization was run on

average: folder containing figures from the parameterization with averaged acute data

average_wPriors: folder containing figures from the parameterization with averaged acute
				 data with log-normal prior
				 
comparisons: folder containing figures that compare various parameterizations

mouse3: folder containing figures from the parameterization with Mouse 3 data

mouse6: folder containing figures from the parameterization with Mouse 6 data

mouse11: folder containing figures from the parameterization with Mouse 11 data
